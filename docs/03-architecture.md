It is responsible for bootstrapping and configuring your application, as well as defining the middleware, console commands, and routes.\n\nYour application's `Application` class can be found at `src/Application.php`. It defines the following hook methods:\n\n- `bootstrap()`: Used to initialize plugins, and attach global event listeners.\n- `routes()`: Used to define your application's routes.\n- `middleware()`: Used to define your application's middleware stack.\n- `console()`: Used to define your application's console commands.\n\n### Bootstrapping your Application\n\nCakePHP provides two places to add bootstrapping logic to your application:\n\n1.  **`config/bootstrap.php`**: This file is included before every request and CLI command. It is ideal for low-level concerns like defining constants, and configuring logging.\n2.  **`Application::bootstrap()`**: This hook method is called after `config/bootstrap.php` is included. It is ideal for loading/initializing plugins, and attaching global event listeners.\n\n```php\n// in src/Application.php\nnamespace App;\n\nuse Cake\\Http\\BaseApplication;\n\nclass Application extends BaseApplication\n{\n    public function bootstrap(): void\n    {\n        // Call the parent to `require_once` config/bootstrap.php\n        parent::bootstrap();\n\n        // Load MyPlugin\n        $this->addPlugin('MyPlugin');\n    }\n}\n```\n\n---\n\n<nav style=\"background: var(--bg-secondary); border: 1px solid var(--border-color); border-radius: 6px; padding: 15px 20px; margin: 30px 0;\">\n  <div style=\"display: flex; align-items: center; justify-content: space-between; flex-wrap: wrap; gap: 10px;\">\n    <a href=\"02-installation-guide.html\" style=\"color: var(--link-color);\">\u2190 Previous: Installation</a>\n    <span style=\"color: var(--text-secondary);\">\ud83d\udcda Page 3 of 3</span>\n    <a href=\"index.html\" style=\"color: var(--link-color);\">Home \u2192</a>\n  </div>\n</nav>\n\n---\n\n**Released under the MIT License.**\n\n**Copyright \u00a9 Cake Software Foundation, Inc.
